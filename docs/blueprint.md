# **App Name**: AlgoShunt

## Core Features:

- Infix Expression Input: Accept user input for infix expressions and initiate the visualization.
- Shunting Yard Visualization: Visually represent the Shunting Yard Algorithm, showing the token stream, operator stack, and output queue with animated transitions.
- Step-by-Step Controls: Provide controls to step forward, backward, auto-play, and reset the visualization process.
- Explanation Panel: Dynamically update an explanation panel with a textual description of the algorithm's steps.
- Postfix & Prefix Conversion: Display the final postfix expression and allow toggling to show the prefix expression as well.
- Teacher Mode: AI powered explanation pop-ups simulating a teacherâ€™s narration, with dynamic highlights that serves as a tool to provide context with each animated move
- Theming and Responsiveness: Support light/dark themes and ensure the application is fully responsive across devices.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to represent algorithms and structure.
- Background color: Very light grey (#F5F7FA) to provide a clean and modern backdrop.
- Accent color: Vibrant orange (#FF6B6B) to highlight critical steps and interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral feel. Code Font: 'Source Code Pro' for infix expression input/output.
- Use Framer Motion for smooth, playful animations of tokens moving through the visualization.
- Visually reference layouts similar to VisuAlgo and AlgoExpert for intuitive interaction.
- Employ a clean, modern icon set to visually represent different operations and functionalities.
